# 🚀 Smart Task Manager

## 📌 Overview
Smart Task Manager is a full-stack web application designed to help individuals and teams efficiently manage tasks, projects, and productivity.

It supports:
- 👤 Individual users (personal task tracking)
- 👥 Teams (task assignment & collaboration)

### Tech Stack
- Frontend: Angular  
- Backend: ASP.NET Web API  
- Database: SQL Server  

---

## 🎯 Problem Statement
Traditional tools:
- ❌ Too simple (e.g., Google Tasks)
- ❌ Too complex (e.g., Jira)

👉 This project provides:
- Task prioritization  
- Deadline tracking  
- Team collaboration  
- Productivity analytics  

---

## ✨ Features

### 🧩 Task Management
- Create tasks with:
  - Title, description
  - Priority (Low → Critical)
  - Due date
  - Categories
- Subtasks support
- Task templates

---

### 📁 Project Organization
- Group tasks into projects
- Color tags & deadlines
- Easy navigation

---

### 👥 Team Collaboration
- Assign tasks
- Comment threads
- File attachments
- @mentions & notifications
- Task watchers

---

### 📊 Task Workflow
- Kanban board:
  - To Do → In Progress → Review → Done
- Priority filtering & sorting
- Due date alerts
- Recurring tasks
- Time logging

---

### 📈 Analytics & Reports
- Personal dashboard
- Productivity reports
- Team performance insights
- Streak system (gamification)
- Export to CSV / PDF

---

## 🏗️ System Architecture

Frontend (Angular)  
↓  
REST API (ASP.NET Web API)  
↓  
Database (SQL Server)
